Cusco GeoNode / About / API

The Cusco GeoNode is a geospatial platform provided by the U.S. Department of State, Humanitarian Information Unit as part of the Secondary Cities project. The Cusco GeoNode is used for collaborating on and sharing geospatial data among the US Government and partners in Peru.

Developers can interact with the Cusco GeoNode through 3 APIs: CSW, OGC Services, and the internal GeoNode API.

Cusco GeoNode uses PyCSW to produce standardized CSW metadata services.

The CSW endpoint is: /catalogue/csw. To retrieve metadata records run something like this: /catalogue/csw?request=GetRecords&service=CSW&version=2.0.2&resultType=results&maxRecords=1000&elementsetname=full&typenames=csw:Record&outputschema=http://www.isotc211.org/2005/gmd. This endpoint can be used by other systems to automatically harvest records from the Cusco GeoNode.

All layers in the Cusco GeoNode are accessible via OGC Services. The WMS Capabilities document is accessible with: /geoserver/wms?SERVICE=WMS&REQUEST=GetCapabilities. This endpoint can be used by geospatial analysts to pull layers directly into their desktop environment (/docs/integration.html). OGC services can also be used to pull layers directly from the Cusco GeoNode into web maps to make mashups.
